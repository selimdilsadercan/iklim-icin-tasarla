Design for Climate: A Full-Stack AI Learning Platform for Middle School Climate Education

Web & mobile platform integrating AI chat personas, teacher analytics, secure student data isolation, and asynchronous pedagogical evaluation.
Student: Selim Dilşad Ercan (150210104)
Advisor: Assoc. Prof. Dr. Gökhan İnce
Department: Computer Engineering Department
University: Istanbul Technical University

Problem

Middle school climate education relies on static materials with limited personal feedback. Research-stage AI models and evaluation scripts are also difficult to deploy securely in real school environments.

Proposed Platform

A full-stack web and mobile learning platform that connects students with climate-focused AI personas, providing teachers with secure analytics and pedagogical evaluation dashboards.

Engineering Challenges Solved

  • Mobile CORS Handling: Capacitor client support for native mobile environments.
  • Long-Running Evaluation: Async queue prevents HTTP gateway timeouts.
  • Student Data Privacy: RLS policies enforce role-based access control.
  • Bulk School Onboarding: CSV/Excel import and credential PDF generation.
  • Real-Time AI Chat: SSE streaming for low-latency conversational interaction.
